WHAT IS THIS PROJECT:
- CRM for FY Intech Solution Sdn Bhd (Malaysian VR/AR/MR/XR company)
- Frontend: React 19 + Vite, hosted on Vercel (auto-deploy from GitHub)
- Backend: FastAPI + SQLAlchemy, hosted on Render (manual or auto-deploy)
- Database: SQLite local / PostgreSQL production
- AI: DeepSeek API for lead generation & outreach drafting
- Web Search: Firecrawl API for company discovery & contact scraping
- Email: Resend API for notifications
- N8N: Original automation workflow (now replaced by CRM's lead_engine.py)

ROUTES:
  /             -> Intro3D splash (4 second 3D animation)
  /login        -> Admin login (default: admin@fyintech.com / admin)
  /dashboard    -> Protected dashboard with 5 tabs:
                    - Dashboard (KPIs + charts)
                    - Active Projects (CRUD for VR/AR projects)
                    - Lead Radar (table of leads, detail modal, edit)
                    - Pipeline (Kanban board, drag & drop)
                    - Settings (profile, notifications, Lead Engine)

LEAD ENGINE (AI-Powered Lead Generation):
  - 12 industries on hourly schedule (2 AM - 1 PM MYT)
  - Pipeline: Firecrawl search -> scrape contacts -> DeepSeek AI extract -> draft outreach -> save to CRM
  - Manual controls in Settings:
    * Run Lead Engine (single industry, dropdown picker)
    * Sweep All Industries (all 12 immediately, async with live progress)
    * Enrich Existing Leads (backfill missing AI data on existing leads)
  - Deduplication by company name & website URL
  - "skipped" = duplicate found, "created" = new lead added
